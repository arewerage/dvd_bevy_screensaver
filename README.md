## Why?

A simple project that shows the basic concepts of the [Bevy Engine](https://github.com/bevyengine/bevy).

Features:
 - Loading images
 - Creating and moving entities
 - Limitation of movement by window size
 - Event handling

## Launch
```
cargo run
```
